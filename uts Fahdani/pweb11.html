Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan elemen-elemen HTML. Dengan CSS, kita dapat mengatur warna, ukuran, posisi, font, dan banyak aspek visual lainnya pada halaman web.

Contoh Penggunaan

Pada halaman ini, kita telah menggunakan berbagai properti dasar CSS seperti background-color, color, font-family, margin, padding, text-align, border-radius, dan lainnya.
